## Comandos Ramas 1

*Ver listado de ramas:*
~~~
git  branch
~~~

*Crear una rama:*
~~~
git  branch  nombre_rama
~~~

*Cambiarnos a una rama:*
~~~
git  checkout  nombre_rama
~~~

*Crear una rama y moverse en un paso:*
~~~
git  checkout  -b nombre_rama
~~~

*Comparar ramas:*
~~~
git  diff  nombre_rama .. nombre_rama
~~~
