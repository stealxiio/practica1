## Comandos básicos 6

*Listar el contenido del repositorio de git:*
~~~
git ls-tree  master
git ls-tree  master ^^^
git ls-tree  master ~3
~~~

*Log en una línea:*
~~~
git  log  --oneline
~~~

*Log con los tres últimos commits en una línea:*
~~~
git  log  --oneline  -3
~~~

*Para más opciones consultar documentación de git.*
