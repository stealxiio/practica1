## Comandos básicos 2

*Ayuda del listado anterior:*
~~~
git  help  log
~~~

*Listar los 5 commits más recientes:*
~~~
git  log -n 5
~~~

*Listar los commits desde una fecha:*
~~~
git  log  --since =2018 -09 -18
~~~

*Listar los commits por autor:*
~~~
git  log  --author =" Antonio"
~~~

*Ver cambios en el directorio:*
~~~
git  status
~~~
